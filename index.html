Then this plugin is for you. Why risk losing page value from tag cloud links to noindexed pages. This simple plugin will insert rel="nofollow" to WordPress Tag Cloud links for SEO reasons. That is all the plugin does. GL Tag Cloud Nofollow should work with any Tag Cloud plugin that correctly uses the wp_tag_cloud function. == Installation == 1. Upload the plugin to your 'wp-content/plugins' directory, or download and install automatically through your admin panel. 2. Activate the plugin through the 'Plugins' menu in WordPress. 3. No other configuration options exist 4. Clear all caches.
